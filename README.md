# NewsManagement

## 1.配置说明:<br>
* 数据库： Mysql 8.0.19  
* IDE: Idea2019
* 框架：SSM(spring springMVC Mybatis)
* 项目管理工具：Maven
* 本地运行需要需改动的配置文件：
